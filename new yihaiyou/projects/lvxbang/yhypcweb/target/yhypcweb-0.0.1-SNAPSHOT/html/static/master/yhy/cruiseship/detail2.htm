    温馨提示：
    1、邮轮会在启航前一小时关闸，关闸后禁止登船，请务必根据办理登船手续时间提前抵达码头办理登船手续，在规定办理登船时间内上船。
    2、以上行程为参考行程，在出发前或航程期间，邮轮公司有权根据天气、战争、罢工、等不可抗力因素调整或改变行程。
    3、请自备牙膏、牙刷、毛巾和拖鞋；每天活动详情请留意船上刊物。
    4、客人信息一旦确认，一律不予以更改。如遇特殊情况需要修改信息（包括客人中英文姓名、性别、出生年月日），收取200元每人的更名费。名单提交后，房内至少有1名客人的名字不能更改，否则视为取消。开航前10天内（含第10天）不接受任何形式的游客信息变更。
    5、邮轮抵离时间及推荐行程内容仅供参考，请以邮轮公司给您的确认为准。若乘客误船，需自行承担相应责任。
    6、在出航前或航程期间，邮轮公司有权根据不可抗力（如气候、天灾、战争、罢工等）取消或变更行程。
    7、船上消费只收取美金，船上提供货币兑换服务，汇率可能略高于国家对外公布的当日汇率。船上不能使用银联卡，只能使用可刷取美金的信用卡，如visa 维萨卡、master 万事达卡、amex 美国运通卡等。
